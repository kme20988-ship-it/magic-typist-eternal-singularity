**Magic Typist: Eternal Singularity** は、タイピングの疾走感と放置ゲームの圧倒的なインフレ体験を融合させた、次世代のサイバーパンク・タイピングRPGです。

## 🌟 主な特徴

-   **⚡ 超速タイピングバトル**: モンスターの属性に合わせて魔法を選択し、詠唱（タイピング）を完遂して撃破せよ。
-   **🦀 Rust & WebAssembly 搭載**: 超高速な計算エンジンをWasmで実装。放置収益やダメージ計算が驚くほどスムーズ。
-   **🐉 ドラゴン進化システム**: 転生と強化を繰り返し、使い魔を「超越神龍」へと進化させよ。
-   **🌍 4つの多元世界**: GENESIS, CYBERIA, ETERNITY, LEGEND。移り変わる天候とギミックがプレイヤーを待ち受ける。
-   **🏆 実績 & 遺物集め**: 8つの「実力証明」と強力な「古代遺物」をコンプリートし、究極のハッカーを目指せ。
-   **🎨 圧倒的ビジュアライザー**: ネオン輝くサイバーパンクな世界観、クリティカルカットイン、そして没入感を高めるBGM。

## 🕹 遊び方

1.  **詠唱（タイピング）**: 画面に表示される魔法属性をキーボード（Q, W, E, R, T, Y）で素早く入力します。
2.  **マニュアル・バースト**: 特定のコンボを達成すると強力なバーストモードが発動。ダメージが爆増します。
3.  **転生（Rebirth）**: 行き詰まったら「霊珠」を獲得して転生し、能力を永続的に底上げしましょう。
4.  **召喚（Gacha）**: 世界中に散らばる強力なユニットを仲間にし、放置収益を最大化させます。

## 🛠 技術スタック

-   **Frontend**: React (Vite), Tailwind CSS, Lucide React
-   **Game Engine**: Rust compiled to WebAssembly
-   **Icons & Assets**: AI generated high-fidelity assets
-   **Animations**: Framer Motion inspired CSS animations

## 🚀 セットアップ

```bash
# 依存関係のインストール
npm install

# Wasmのビルド (was-packが必要)
npm run build:wasm

# 開発サーバーの起動
npm run dev

# プロダクションビルド
npm run build
```
